title: Bookmarks Widget
description: Save and organize your favorite websites in one place
---

The Bookmarks widget lets you save and organize links to your favorite websites with automatic favicon detection and drag-and-drop reordering.

## Features

- **Quick Access**: Save frequently visited websites
- **Favicon Display**: Automatic logo/icon detection for popular sites
- **Drag & Drop**: Reorder bookmarks by dragging
- **Link Management**: Add, edit, and delete bookmarks
- **One-Click Open**: Click any bookmark to open in a new tab
- **Visual Organization**: Icons make bookmarks easy to identify

## Setup

No integration required—the widget works out of the box!

## Adding Bookmarks

<Steps>
  <Step title="Click Plus Icon">
    Click the + button in the widget header
  </Step>
  
  <Step title="Enter Title">
    Give the bookmark a descriptive name
  </Step>
  
  <Step title="Enter URL">
    Paste the full website URL (including https://)
  </Step>
  
  <Step title="Save">
    Click "Add link" to save the bookmark
  </Step>
</Steps>

## Usage

### Bookmark Display

Each bookmark shows:
- **Favicon/Logo**: Automatically detected site icon
- **Title**: Your custom bookmark name
- **Link Icon**: Indicates it's a clickable link
- **Delete Button**: Appears on hover

### Opening Links

Click any bookmark to:
- Open the link in a new browser tab
- Preserve your current dashboard view

### Reordering Bookmarks

Change bookmark order with drag and drop:
1. Click and hold on a bookmark
2. Drag it to the desired position
3. Release to drop it in place
4. Order saves automatically

**Touch Support**: Long-press on mobile/tablet to start dragging.

### Deleting Bookmarks

<Steps>
  <Step title="Hover Over Bookmark">
    Move your cursor over the bookmark you want to remove
  </Step>
  
  <Step title="Click Delete">
    Click the trash icon that appears
  </Step>
  
  <Step title="Confirm">
    The bookmark is removed immediately (no confirmation dialog)
  </Step>
</Steps>

## Favicon Support

The widget automatically detects favicons for popular websites:
- **Social Media**: GitHub, Twitter, Facebook, LinkedIn, etc.
- **Development**: Stack Overflow, GitLab, npm, etc.
- **Productivity**: Google services, Notion, Slack, etc.
- **General**: Most modern websites with favicons

If a favicon isn't detected, a generic link icon is shown.

## URL Validation

Bookmarks must include:
- Valid URL format (checked automatically)
- Protocol (http:// or https://)
- Domain name

Example valid URLs:
- `https://github.com`
- `https://www.notion.so/workspace`
- `https://tryforge.io`

<Warning>
  URLs without a protocol (e.g., `github.com`) will fail validation. Always include `https://` or `http://`.
</Warning>

## Drag & Drop

### Mouse Control

- **Activation**: Click and drag (10px threshold to start dragging)
- **Visual Feedback**: Dragged item becomes semi-transparent
- **Drop Zones**: Vertical list with clear drop positions

### Touch Control

- **Activation**: Long-press for 250ms
- **Tolerance**: 5px movement tolerance
- **Mobile-Optimized**: Works smoothly on phones and tablets

## Organization Tips

**Group by Category**: Order bookmarks by type (work sites, social media, tools, etc.)

**Most Used First**: Put frequently accessed sites at the top

**Clear Naming**: Use descriptive titles so you can scan quickly

**Regular Cleanup**: Remove outdated or unused bookmarks

## Storage

Bookmarks are stored:
- In widget configuration
- Saved automatically on every change
- Synced with your Forge account
- Persisted across devices

Each bookmark contains:
- **ID**: Unique identifier (auto-generated)
- **Title**: Your custom name
- **Link**: Full URL

## Troubleshooting

<Warning>
  If bookmarks aren't saving:
  1. Check your internet connection
  2. Verify the URL format is correct
  3. Try refreshing the page
  4. Contact support if the issue persists
</Warning>

### Common Issues

**Invalid URL error**: Make sure to include `https://` or `http://` in the URL.

**Drag not working**: Ensure you're clicking and holding (or long-pressing on mobile) before dragging.

**Missing favicon**: Some websites don't provide favicons or block external access to them.

**Link not opening**: Check that the URL is correct and the site is accessible.

## Limits

While there's no hard limit on the number of bookmarks:
- Performance is optimal with up to 50 bookmarks
- The widget scrolls if you have more bookmarks than fit
- Consider using multiple bookmark widgets for different categories

## Widget Sizes

- **Desktop**: 1 column × 2 rows
- **Tablet**: 1 column × 2 rows
- **Mobile**: 1 column × 1 row

</Info>
