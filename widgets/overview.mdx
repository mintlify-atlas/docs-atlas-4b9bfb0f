## Widget Features

### Responsive Design
All widgets are designed to adapt to different screen sizes (desktop, tablet, mobile) with optimized layouts for each breakpoint.

### Drag & Drop
Many widgets support drag-and-drop functionality for reordering items, moving tasks between columns, or organizing content.

### Real-time Updates
Widgets that connect to external services automatically refresh to show the latest data, with configurable refresh intervals.

### Customization
Most widgets offer configuration options such as:
- Filtering and sorting
- Custom timeframes
- Color themes
- Display preferences

### Integration Support
Widgets can integrate with popular services:
- **Google** (Calendar, Gmail)
- **GitHub** (Repositories, Issues, PRs)
- **Notion** (Pages and notes)
- **Coinbase** (Cryptocurrency prices)

## Adding Widgets

To add a widget to your dashboard:

<Steps>
  <Step title="Open Widget Gallery">
    Click the "Add Widget" button on your dashboard
  </Step>
  
  <Step title="Select Widget">
    Browse available widgets and select the one you want to add
  </Step>
  
  <Step title="Configure Integration (if required)">
    Some widgets require connecting to external services like Google or GitHub
  </Step>
  
  <Step title="Customize Settings">
    Adjust widget-specific settings like filters, timeframes, or display options
  </Step>
</Steps>

<Note>
  Widget configurations are saved automatically and persist across sessions.
</Note>
