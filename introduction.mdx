# Welcome to Forge

Forge is a new way of building dashboards. Instead of juggling multiple websites and programs, Forge brings all your most important tools into a single, customizable workspace. Track stock prices, check the weather, monitor GitHub activity, manage tasks, or write in Markdown—all from one place.

## Available Widgets

Forge includes 14 powerful widgets to enhance your productivity:

- **Meetings** - View your upcoming Google Calendar events and create new meetings
- **Inbox** - Monitor your Gmail inbox with label filtering and real-time notifications
- **GitHub** - Track issues, pull requests, and repository activity
- **Crypto** - Monitor cryptocurrency prices with real-time charts via Coinbase
- **Weather** - Get local weather forecasts with hourly and daily views
- **Markdown Editor** - Draft notes and documentation with a rich text editor powered by Tiptap
- **Bookmarks** - Save and organize your favorite links in one place
- **Todo** - Manage tasks with checkboxes, drag-and-drop, and reminders
- **Clock** - Display the current time with timezone support
- **Countdown** - Track time until important events with customizable countdowns
- **GitHub Heatmap** - Visualize your GitHub contribution history
- **Kanban** - Organize tasks in a visual board with columns and cards
- **Frame** - Embed external websites and web applications directly in your dashboard

## Tech Stack

Forge is built with modern, production-ready technologies:

- **Framework**: Next.js 16 with React 19
- **Styling**: Tailwind CSS 4 with custom theming
- **Database**: PostgreSQL with Drizzle ORM
- **Real-time**: Upstash Redis for notifications and pub/sub
- **Authentication**: BetterAuth with OAuth support
- **Validation**: Zod for type-safe schemas
- **Data Fetching**: TanStack React Query for caching and mutations

## Use Cases
